# 🧮 會計系學生的分錄練習題輔助工具

本專案專為會計系學生設計，旨在解決備考期間「大量手寫分錄」所帶來的體力消耗與時間浪費。透過數位化輔助工具，學生能將心力專注於**解題邏輯的磨練**與**會計觀念的運用**，而非重複的書寫勞動。

此外，本工具整合了題目截圖與 PDF 匯出功能，能協助學生快速建立個人化的「**錯題本**」，實現數位化保存與考前的高效率複習。

## 🌟 核心功能
* **快速科目搜尋**：透過關鍵字篩選常用會計科目（開發中）。
* **智慧金額輸入**：支援「雙擊複製、單擊貼上」功能，提升輸入效率。
* **自動生成備註**：自動加總計算過程，並附帶 `💡 計算過程：` 標示，方便覆核。
* **互動式編輯**：生成後的日記簿內容支援動態修改（日期、科目、金額）。
* **錯題本生成**：支援將練習成果匯出為 PDF 格式，方便學生收集弱點題目。
* **版本控制**：專案透過 Git 進行管理，確保開發過程安全可追溯。

## 🛠️ 技術架構
* **Frontend**: HTML5, JavaScript (ES6+), Tailwind CSS (CDN版本)
* **Library**: jsPDF / html2canvas (預計用於 PDF 導出)
* **Version Control**: Git / GitHub
* **Editor**: VS Code

## 🚀 如何使用
1. 複製此專案至本地端或直接開啟 `index.html`。
2. 在左側選擇會計科目（搜尋功能優化中）。
3. 輸入對應金額，系統將自動處理借貸平衡提示。
4. 點擊「產生分錄」即可獲得完整格式。
5. 使用「導出 PDF」功能保存你的練習紀錄。
