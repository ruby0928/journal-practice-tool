# 🧮 會計分錄練習工具

專為會計系學生設計的數位輔助工具，將心力專注於**解題邏輯**與**會計觀念的運用**，而非重複的書寫勞動。整合題目截圖、公式計算、自動備註與 PDF 匯出，快速建立個人化的「錯題本」。

## ✨ 功能特色

### 科目庫
- 內建中級會計學等級科目（資產、負債、權益、損益四大類）
- 關鍵字即時搜尋，自動展開符合的類別
- 可新增自訂科目至科目庫（附刪除按鈕），或建立僅本次使用的暫時科目
- 科目拖曳至借貸方後，可直接點擊修改名稱

### 金額輸入
- 輸入 `=算式` 後按 Enter，自動計算並格式化（支援 `+` `-` `*` `/`）
- 離開欄位時自動補上千位分隔符
- **雙擊**已輸入的金額欄複製數值（含公式）；**單擊**空白欄貼上

### 分錄生成與編輯
- 點擊「＋ 新增分錄」產生帶日期的日記簿區塊
- 日期支援「月／日」與「第N季」兩種模式切換
- 產生後仍可：
  - 直接點擊科目名稱或金額修改
  - 新增借方／貸方科目列
  - 拖曳調整科目順序
  - 刪除單一科目列
  - 複製整個分錄區塊
- 手動新增科目後，顯示即時借貸平衡提示（2.5 秒後自動消失）

### 計算過程備註
- 使用公式輸入時，自動在「💡 計算過程」區塊產生計算備註
- 相同公式的備註會自動合併科目名稱（例如：`應收帳款、應付帳款: 1,000*5 = 5,000`）
- 備註可拖曳排序、手動刪除，或透過「＋ 新增備註內容」在方塊下方直接輸入補充說明
- **雙擊備註文字**即可直接修改內容，點擊其他區域自動儲存
- 可自行選擇列印時是否包含計算過程；備註欄位為空時列印自動省略分隔線與標題

### 其他
- **題目截圖區**：Ctrl+V 貼上試題圖片，隨分錄一起匯出
- **自動暫存**：所有編輯狀態自動存至瀏覽器（localStorage），關閉視窗後重開可接續
- **一鍵清除**：清除全部內容並重置（需手動確認）
- **PDF 匯出**：分錄過多時自動換頁，每個分錄保持完整不切割；題目圖佔滿第一頁全寬，後續頁直接接分錄

## 🚀 使用方式

直接用瀏覽器開啟 `index.html`，無需安裝或伺服器。

```bash
# 若瀏覽器限制 file:// 協定，可啟動本地伺服器
python -m http.server 8080
# 瀏覽 http://localhost:8080/
```

## 🛠️ 技術

- HTML5 / Vanilla JavaScript / Tailwind CSS (CDN)
- [mathjs 12.4.1](https://mathjs.org/) — 公式計算
- [jsPDF 2.5.1](https://github.com/parallax/jsPDF) + [html2canvas 1.4.1](https://html2canvas.hertzen.com/) — PDF 匯出
- [Sortable.js 1.15.0](https://sortablejs.github.io/Sortable/) — 拖曳排序
